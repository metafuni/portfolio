Geography Now! e-learn Application

A tribute e-learn application for the Geography Now! Youtube channel. The application is aimed for people wanting to explore the world and improve their geography knowledge, but can certainly be used at high school in geography classes as a fun intermezzo. The user can use the three functionalities of the app: Explore, GeoMap and the Quiz.

In Explore the user can look up any country of the world and learn about the local languages, the location in the world, currency used and the living standards of the capital cities.

The GeoMap is a map of the world with all the capital cities marked. Clicking on the capital cities it will pop up the Geography Now! episode of the country.

The user can test its geography knowledge by playing the Quiz. The goal of the quiz is to guess all the countries of the world within the time limit.

  • React
  • JavaScript
  • SCSS
  • Mapbox API

Chef Metafuni Jamstack Website

A personal website including a blog, pictures gallery and recipes from the Instagram channel @chefmetafuni.

The website is built on a Jamstack using Gatsby, as a static site generator that works on the JavaScript framework React, the Contentful CMS as a back-end to provide and update content and GraphQL to query the data from the headless CMS. The benefits for using this stack are simply to have a: a fast single-page application (SPA), better overall performance, optimized images and content for better loading speed and integration with a headless CMS as back-end for easy and convenient content delivery.

  • Gatsby
  • GraphQL
  • Bootstrap
  • ContentfulCMS

The Movie Database (TMDb) movie search application

A movie database application to look up your favourite movies and documentaries. The React application uses the TMDb API to fetch the data and movie posters, and the Youtube API to fetch the trailer for the movie. The autosuggest in the search field is done using the easy to use React Autosuggest component.
